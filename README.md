# Excel Data Analysis Project: Amazon Sales 2025 INR

## 1. Project Objective
 "Analyze sales data to find, top products, monthly sales and insights for decision-making"

## 2. Tools Used
- Excel
- Pivot Tables
- Charts
  
## 3. Dataset
- Source of data - https://www.kaggle.com/datasets/anandshaw2001/amazon-product-sales-2025 
  
## 4. Steps Performed
1. **Data Cleaning** – removed empty rows, formatted columns, fixed errors.  
2. **Pivot Table Creation** – analyzed Customer reviews, Delivery Analysis, Top product Sales, Monthly Sales 

## 5. How to Use
1. Open the Excel file in `Excel-Files/`  
2. Explore Pivot Tables and charts
